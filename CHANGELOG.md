# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-30

### Added

- Initial release of CRTR package
- Instagram reel downloading functionality
- Audio extraction from video files
- Audio transcription using Whisper AI
- Recipe generation using Google Gemini AI
- Structured JSON output with nutritional analysis
- Danish language support with metric measurements
- Command-line interface (CLI)
- Comprehensive documentation
- Unit tests for core functionality

### Features

- Support for multiple Instagram URL formats
- Automatic device detection (CUDA/CPU) for transcription
- Multiple Whisper model sizes (tiny to large-v3)
- Multiple Gemini AI models
- Automatic cleanup of temporary files
- Error handling and logging
